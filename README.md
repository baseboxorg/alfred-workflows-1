# Alfred Workflows

Make your [Alfred App](http://www.alfredapp.com) more powerful. Workflow for
[Homebrew](./hombrew) and
[RubyGems](./rubygems).

## Dependencies

* [Ruby](http://www.ruby-lang.org)
* [Alfred](http://www.alfredapp.com)
* [Alfred Powerpack](http://www.alfredapp.com/powerpack/)
